== README

LPG intranet mockup.
  - User base sign-in/register with google-oauth(omniauth) & devise
  - News/Event postings (editable by AdminUser)
  - Employee listing
  - All in Google's Material Design with Polymer

Gems:

  * gem 'devise'
  * gem 'omniauth-google-oauth2'
  * gem 'bower-rails'
  * gem 'emcee'
